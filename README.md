# プッシュアップの記録をするサイトです。

（フォームの欄から通常、プッシュアップバーあり、デクラインを選べます）

筋肉増量や筋力アップには色々なアプローチがあると思いますが、私はシンプルに回数を追って行くのが一番良いアプローチだと考えています。
追い込み論や総ボリューム理論などアプローチの仕方は色々あると思いますが、私は出来る回数が多くなればその分筋肉が付くと考えています。

※勿論回数のみを追うのではなく重さを足したり、工夫して負荷を増やしたりするという観点も勿論大切ですが、今回は自重（自体重トレーニング）のプッシュアップにのみ絞った話とします。

そこで回数を記録にしていき、折れ線グラフとして見ていけば上り調子の時や逆に下がり調子の時があるということを後から見ることが出来ます。
（通常、プッシュアップバーあり、デクライン、それぞれフォーム別に折れ線グラフを用意してます）

そして上り調子の時はどんなトレーニングをしていたか、下がり調子の時はどうだったのか、それぞれの期間での取り組み方はどう違うのか
食事が足りなかったのか、回復が足りなかったのか等のヒントが得られるかもしれないと考えてます。

そんな感じの簡易的な記録サイトですが、とりあえず作ってみました。
メモに回数を記録したりしても事足りるとは思うのですが、多少分かりやすく記録が出来るんじゃないかと思います。

# このサイトでできること

日付・回数・フォームを入力して記録

記録データはブラウザの localStorage に保存

フォーム別（通常 / バーあり / デクライン）に折れ線グラフを表示

全フォーム合計の推移も確認可能

日付ごとの記録をまとめて削除

フィルタでフォーム別に表示切り替え

# トレーニングの振り返り

調子が良かった期間にどんな取り組みをしていたか

回数が落ち込んだ時期に、食事や休養がどうだったか

フォーム別の得意不得意や偏り

トレーニングサイクルの改善ポイント

# 技術要素

HTML / CSS / JavaScript

Chart.js による折れ線グラフ描画

localStorage でのデータ保存

シンプルなバニラ JavaScript 設計

# 最後に

日々の変化を見える化することで、自分のトレーニング傾向を振り返りやすくなるはずです。
